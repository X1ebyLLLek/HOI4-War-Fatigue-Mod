[ events/plus_cs_optimization_loop.txt ] (Ивент: plus_cs.999)

       |
       |───[ РАСЧЕТ ИСТОЩЕНИЯ И ДАВЛЕНИЯ ]───────────────────────────────────────────────────────────────.
       |    |                                                                                            |
       |    v                                                                                            |
       |   [ scripted_effects/plus_cs_crisis_checks.txt ] -> Скрипт: plus_cs_crisis_step                 |
       |    |  (Проверяет базовую выживаемость: законы экономики, модификаторы Total Exhaustion)         |
       |    v                                                                                            |
       |   [ scripted_effects/plus_cs_war_exhaustion.txt ] -> Скрипт: plus_cs_update_exhaustion_vars     |
       |    |  (Математика соотношения потерь к фабрикам: высчитывает plus_cs_pressure_ratio)            |
       |    |                                                                                            |
       |    |──(Давление: Среднее) --> Флаг: plus_cs_war_strain                                          |
       |    |──(Давление: Высокое) --> Флаг: plus_cs_exhausted ----------------------------------.       |
       |    └──(Давление: Полное ) --> Динамический Модификатор: total_exhaustion                |       |
       |                                                                                         v       |
       |                   [ events/plus_cs_internal.txt ] (ВНУТРЕННИЕ КРИЗИСЫ) <----------------'       |
       |                    ├── plus_cs_internal.1: Ультиматум Генералов (Бунт в армии)                  |
       |                    ├── plus_cs_internal.10: Политический Коллапс (Гражданская война)            |
       |                    └── Влияние на марионетки: [ events/plus_cs_puppet_events.txt ]              |
       |                        (Скрипты plus_cs.300: Марионетки бросают падшего хозяина)                |
       |                                                                                                 |
       |───[ ПУЛЬС ИИ (Ежемесячный генератор дипломатии Бота) ]──────────────────────────────────────────'
            |
            v
           [ events/plus_cs_ai_peace_pulse.txt ] -> Ивент: plus_cs_pulse.1
           Раз в месяц ИИ бросает кости, учитывая математику истощения (плюс черты лидеров-фанатиков).
            |
            ├── (Вероятность 1) --> [ events/plus_cs_white_peace_event.txt ] (Ивент: plus_cs.4)
            |                        Бот готов к Белому Миру. Отправляет запрос напрямую врагу.
            |
            ├── (Вероятность 2) --> [ events/plus_cs_submission_events.txt ] (Ивент: plus_cs.201)
            |                        Бот полностью разбит. Предлагает стать марионеткой или сдать клаймы без войны.
            |
            └── (Вероятность 3) --> Бот инициирует сложную Сдачу (conditional surrender).
                                     (Переход к ВЕТВИ "ОТПРАВКА ОФФЕРА")

[ ДО-ВОЕННАЯ ДИПЛОМАТИЯ И ТЕРРИТОРИИ ]: (УЛЬТИМАТУМЫ И СДЕЛКИ ДО НАЧАЛА БОйНИ)

[ events/plus_cs_territory_demands.txt ]
Агрессор требует коренные земли у слабого соседа (Бесплатные клаймы).
 |
 ├── Сосед Отказывается -> Немедленное Объявление Войны (Wargoal Activated)!
 └── Сосед Соглашается -> Территории передаются. Агрессор получает плюс к стабильности.

[ events/plus_cs_diplomacy.txt ]
Смарт-сделки. Слабая страна сама предлагает агрессору территории до войны, чтобы откупиться.


=============================================================================================================
[ ОТПРАВКА ОФФЕРА ]: ПЕРЕХВАТЧИКИ И ДИКТАТ ЛИДЕРОВ АЛЬЯНСА
=============================================================================================================
Мирное предложение (Сдача, Белый мир) сформировано. Оно летит к Получателю (Врагу).
НО ВМЕШИВАЕТСЯ СУБОРДИНАЦИЯ АЛЬЯНСА!

 [ Игровой движок (мод) проверяет статус Получателя ]
       |
       ├── Получатель - Одиночка (не в альянсе) -----------------.
       |                                                         |
       ├── Получатель - Сам Лидер своего Альянса ----------------|
       |                                                         |
       └── Получатель - Минор в чужом Альянсе (не Лидер)         v
            |                                      (ПЕРЕХОД К ФИНАЛУ: ОКНО МИРА)
            v
           [ events/plus_cs_faction_veto.txt ] (плюс: plus_cs_faction_peace.txt)
           Запрос перехватывается, минору запрещено принимать мир самому.
           Мод замораживает сделку и кидает запрос напрямую Лидеру Фракции (plus_cs.20).
            |
            ├── Лидер ЗА (plus_cs.21) -------> ОДОБРЕНО (ПЕРЕХОД К ФИНАЛУ: ОКНО МИРА)
            |
            └── Лидер ПРОТИВ (plus_cs.22) ---> Лидер применяет вето.
                 |
                 ├── Минор смиряется и послушно продолжает войну как пушечное мясо.
                 └── Минор начинает БУНТ (plus_cs.23) -> Выходит из альянса и подписывает Сепаратный Мир!
                     (Получает модификатор Предателя Альянса).
